#StockProject

**股票价格预测**

以下是关于项目股票预测实验中的一些原理及实验数据记录。主要设计实验中关于LSTM神经网络ESN网络的原理。其中LSTM又有多变量和单变量的区别因此一共有三种神经网络模型。对应的蟒蛇代码在StockPredict /的getData下。
